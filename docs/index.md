## Overview

nblade is a high-performance geometric algebra library implemented in Rust, with Python bindings. It supports:

- Arbitrary dimensions (up to 64D)
- Arbitrary metric signatures G(p, q, r)
- Dense and sparse multivector representations
- All standard geometric algebra operations

## 概述

nblade 是一个基于 Rust 实现的高性能几何代数库，提供 Python 绑定。支持：

- 任意维度（最高 64 维）
- 任意度量签名 G(p, q, r)
- 密集和稀疏多重向量表示
- 所有标准几何代数运算
